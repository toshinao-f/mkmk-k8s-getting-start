# 入門 Kubernetes 読書会
## 概要
「入門 Kubernetes」をの読了を、みなさんといっしょに目指すというものです！
Kubernetesに興味のある方、入門 Kubernetesを買ってしまった方、この機会にいかがでしょうか。
基本的に、当日読む時間（1時間～2時間）を設け、終わりに簡単に感想などの共有会（15分～30分）で進めていこうと考えています。
ただ、進め方は参加者のご都合、ご意見に合わせて変えていこうと思っています。
